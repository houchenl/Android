
# 图片

## ImageView scaleType
``` shell
# 居中显示，不缩放。当图片的长/宽超过View的长/宽，多余部分剪裁掉。
center

# 居中显示，同时按原图比例放大图片，直到图片的宽/高都大于或等于view的宽和高。目标是充满view。
centerCrop

# 居中显示，同时按原图比例缩小图片，直到图片的宽/高都小于或等于view的宽和高。目标是将图片显示完整。
centerInside

# 按原图比例缩放图片，直到图片显示完整，且占满view的宽度或高度中的至少一个。然后居中显示
fitCenter

# 按原图比例缩放图片，直到图片显示完整，且占满view的宽度或高度中的至少一个。然后居上或居左显示
fitStart

# 按原图比例缩放图片，直到图片显示完整，且占满view的宽度或高度中的至少一个。然后居下或居右显示
fitEnd

# 将图片拉伸填满整个view，不按原图比例缩放。
fitXY

# 按图片原始大小，从view左上角开始绘制。如果图片大于view，多余部分剪裁掉，如果图片小于view，不做处理。
matrix
```
