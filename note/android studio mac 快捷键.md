
# android studio mac 快捷键

## 代码格式化
``` shell
# 清除未使用import
ctrl + option + O

# 代码格式化
command + option + L
```

## 代码查找
``` shell
# 查找class
command + O

# 查找标识符
command + option + O
```

## 代码生成
``` shell
# 生成代码，打开generate选项卡
command + N
```
